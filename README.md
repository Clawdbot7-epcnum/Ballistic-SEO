# Ballistic SEO - Firearm Industry SEO Website

## Overview
Professional SEO services website tailored for the firearm industry. Clean, modern design with subtle firearm/tech aesthetics.

## Features Built
✅ **Responsive Design** - Works on desktop, tablet, mobile
✅ **Modern Dark Theme** - Professional with orange accents
✅ **Smooth Animations** - Fade-in effects and hover states
✅ **SEO Optimized** - Proper meta tags and semantic HTML
✅ **Fast Loading** - No external dependencies except Google Fonts
✅ **Industry-Focused** - Copy written specifically for firearm businesses

## Sections Included
- **Hero Section** - Strong headline with key stats
- **Services Grid** - 6 core SEO services for firearm industry
- **About Section** - Industry-specific value proposition
- **CTA Section** - Lead generation focused
- **Footer** - Professional links and contact

## Color Palette
- Primary: `#1a1a1a` (Dark background)
- Secondary: `#2d3748` (Card backgrounds)
- Accent: `#ff6b35` (Orange for CTAs and highlights)
- Text: `#ffffff` (White text)
- Muted: `#a0aec0` (Secondary text)

## Typography
- **Inter** - Clean, modern sans-serif for body text
- **JetBrains Mono** - Monospace for stats and technical elements

## To Customize
1. **Replace placeholder content** in the HTML
2. **Add your actual images** (hero background, service icons, etc.)
3. **Update contact email** in the CTA and footer
4. **Add your logo** in the navigation
5. **Customize colors** in the CSS variables if needed
